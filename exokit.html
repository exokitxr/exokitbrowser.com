Exokit is a browser from the future.
It runs virtual worlds and can't draw HTML. 

npm install exokit

How it works

Exokit is the opposite of a web browser.

Most web browsers run web sites. But Exokit lets the site run the browser! It's an exobrowser!

Exokit is just Javascript, so it can be changed by a site.

More importantly, Exokit is native. So, Exokit executes web sites natively.

This makes Exokit fast. And perfect for Mixed Reality on the web.

The catch? Exokit can't draw HTML.
