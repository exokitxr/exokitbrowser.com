Exokit is a browser from the future.

It runs virtual worlds and works everywhere your web code does. 

npm install -g exokit
exokit http://zeovr.io

WHAT'S IT DO?

Exokit is a fast, full HTML5 web browser, written as a Node.js module.

It supports all the standard browser APIs and accepts keyboard, mouse, and mixed-reality input.

Actually, Exokit can do everything a browser can do -- except render HTML (mostly).

WHAT CAN I BUILD?

Exokit can't render HTML, but it can draw Canvas and WebGL -- natively, and fast.

This means you can code robust AR and VR experiences using your favorite flavor of JavaScript, just like you code a website.

Under the hood, everything is plain OpenGL, so it plays nicely with Windows, Linux, macOS, iOS, Android, and even Magic Leap.
