Exokit is a browser from the future.
It runs virtual worlds and works everywhere your web code does. 

npm install -g exokit
exokit http://zeovr.io

What's it do?

Exokit is a fast, full HTML5 web browser, written as a Node.js module. It supports all the standard browser APIs and accepts keyboard, mouse, and mixed-reality input. Actually, Exokit can do everything a browser can do -- except render HTML (mostly).

What can I build?

Exokit can't render HTML, but it can draw Canvas and WebGL -- natively, and fast. This means you can code robust AR and VR experiences using your favorite flavor of JavaScript, just like you code a website. Under the hood, everything is plain OpenGL, so it plays nice with Windows, Linux, macOS, iOS, Android, and even Magic Leap.

But can we say something in list format?

You bet. Some examples of what Exokit can do:

  • Load any http(s) site
  • Parse a programmatic DOM
  • Run any <script>
  • Load any <image>, <video>, <audio>
  • Register and run Web Workers
  • Render Canvas2D
  • Render WebGL
  • Render WebVR
  • Handle gamepad input
  • Iframe isolation
  • Embed anywhere with node
  • Run on Android/iOS
  • Run tests
  • Power a web bot

And API support, also as a list?

We're on it:

  • HTTP(S)
  • HTML5

  • DOM

  • Image tag
